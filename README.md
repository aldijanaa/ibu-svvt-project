# IBU SVVT Project

Repository for the **Software Verification, Validation & Testing** project (Master), IBU.

This codebase is based on the open-source **Budgie** expense-tracking app. It was copied from the original public repository for use in this course project. Any modifications made here are for academic and testing purposes.

## Installation

```plaintext
git clone https://github.com/aldijanaa/ibu-svvt-project.git
cd ibu-svvt-project
npm install
npm start
```
